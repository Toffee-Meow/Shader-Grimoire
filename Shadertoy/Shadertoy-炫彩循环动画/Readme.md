## Shadertoy-炫彩循环动画

平台：Shadertoy

### 内容

&emsp;&emsp;学习一位油管大佬的作品，实现一个超酷的类似“万花筒”样式的炫彩循环动画Shader效果

### Shader文件

&emsp;&emsp;Shader文件使用GLSL代码

- DazzlingLooping_GLSL.glsl：炫彩循环动画Shader

> PS：GLSL Shader代码，可在Shadertoy网站运行
>
> 或者参考代码实现思路，运用在游戏引擎当中
